# 🚗 Car Rental Dashboard – Power BI

## 📊 Overview
This project presents an interactive Power BI dashboard designed to analyze car rental business performance, including bookings, revenue, and fleet utilization.

## 🎯 Objectives
- Track bookings and revenue performance
- Monitor fleet size and utilization
- Analyze booking status and trends
- Support data-driven business decisions

## 📈 Key Features
- KPI Cards: Revenue ($1.7M), Rentals (1K), New Bookings (96), Fleet Size (50 vehicles)
- Revenue trend analysis with month-over-month comparison
- Booking status distribution (Completed, Ongoing, Booked)
- Revenue by car make (Top performer: Audi)
- Detailed booking table with transaction data
- Unit-level analysis by Make & Model with growth indicators

## 🛠 Tools & Technologies
- Power BI
- DAX
- Data Modeling
- Data Visualization
